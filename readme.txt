Divi Child Theme

A ready to modify bare-bones child theme for use with Divi Theme.

To use this Child Theme, you must have Divi parent theme installed.

For more information about Divi Child Theme please go to:
http://hyperlinkcode.com/blog/divi-child-theme/


For more information about Divi parent Theme please go to:
http://hyperlinkcode.com/blog/divi/

To find matching colors quickly please visit:
http://www.html-color-names.com

==
Installation
==


1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.


2. Click 'Upload Theme'.

3. Click 'Choose File' and browse to divi-child.zip on your hard drive and select it.

4. Once you have selected divi-child.zip, click 'Install Now'

5. Click on the 'Activate' button to use your new theme right away.



6. Navigate to Appearance > Editor in your WordPress admin panel to add custom modifications.
